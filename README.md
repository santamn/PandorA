# 全自動授業資料ダウンロードアプリケーション PandorA

## 概要
PandorAはPandAから現在受講中の授業の資料を4時間おきに確認し、新規に追加されたものがあればダウンロードを行うアプリケーション。

## 使い方
お使いのOSに合わせて以下のリンクからダウンロードしてください。

- Windows
- Mac
- Linux

ダウンロードしたアプリケーションを実行すると、下のようにメニューバーにPandorAボタンが表示されます。
